# Portfolio — Andrei Paulo Gueta

A minimalist, modern portfolio built with **Next.js**, **Tailwind CSS**, and **Framer Motion**. Theme: black, gray, and gold.

## Getting started

```bash
cd web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

- **Profile & copy:** `src/data/site.ts`
- **Projects & screenshots:** `src/data/projects.ts` and `src/lib/images.ts`
- **Asset files:** `public/images/`
- **Social links:** `site.social` in `src/data/site.ts`

## Project screenshots

Place files under `public/images/` (matching folder names). Update paths in `src/lib/images.ts` when adding or renaming screenshots.
